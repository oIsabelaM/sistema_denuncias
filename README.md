# sistema_denuncias
Sistema de denúncias anônimas: 

Este projeto consiste em um site que permite ao usuário realizar denúncias de forma anônima e organizada. O sistema possui categorias de denúncia, campos para localização, envio de evidências (simulado), geração automática de protocolo e acompanhamento do status da denúncia (recebida, em análise, encaminhada e concluída).
